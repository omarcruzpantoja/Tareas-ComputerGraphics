Scenegraph

Description:
This is s simple robot scenegraph created using canvas.

Functionalities:
Press H to open/close Controls
Change control value x0 to change the X position of the robot.
Change control value y0 to change the Y position of the robot.
Change control value a1 to change the rotation angle of the arm (color red)
Change control value a2 to change the rotation angle of the upper leg(color green)
Change control value a3 to change the rotation angle of the lower leg (color blue)
