Asteroid

Turbo time:

Power: INFINITE TURBO ON [OFF]

Description:
Simulation of asteroid game using canvas.
Includes 3 modes and its parameters can be change in the controls section
Objective: Hit the asteroid before it destroys your ship!
Pick the power up(In form of a blue star) to have infinite turbo!
If the asteroid hits the ship, the ship will set in the center
If you hit the asteroid with a laser bullet, it will reappear randomly in the map and you get instant reload!
***Game not completed***

Functionalities:
Press letter H to open/close control setup
Change x0 and y0 parameters to instantly relocate ship into the exact coords given respectively

Mode 0 - In this mode we have an event based movement
Arrow key up: Move forward [Fixed displacement parameter] pixels
Arrow Key down: Move backwards [fixed displacement parameter] pixels
Arrow Key : Rotate ship clockwise [Angle step] degrees
Arrow Key : Rotate ship counter-clockwise [Angle step] degrees
Spacebar: Shoot laser


Model 1- Displacement based on polling with fixed velocity
Arrow key up: Move forward [Fixed velocity parameter] pixels per second
Arrow Key down: Move backwards [fixed velocity parameter] pixels per second
Arrow Key : Rotate ship clockwise [Angle velocity] degrees
Arrow Key : Rotate ship counter-clockwise [Angle velocity] degrees
Spacebar: Shoot laser
Arrow key up + t : move forward with a boost (Progress bar shows turbo time- max 3 seconds)
Pick up power up(star): Will allow the ship to have infinite turbo for 5 seconds


Model 2- displacement based on polling with fixed acceleration
In this mode, the ship will continue to float around the canvas, simulating space. In this game we added an "Atmosphere" resistance, which will cause the ship's velocity to drop slowly.

Arrow key up: Move forward [Fixed acceleration * vx] pixels per second
Arrow Key down: Move backwards [fixed acceleration *vy] pixels per second
Arrow Key : Rotate ship clockwise [Angle velocity] degrees
Arrow Key : Rotate ship counter-clockwise [Angle velocity] degrees
Spacebar: Shoot laser
Arrow key up + t : move forward with a boost (Progress bar shows turbo time- max 3 seconds)
Pick up power up(star): Will allow the ship to have infinite turbo for 5 seconds
