User functionalities: Click on any area of the SVG/Canvas box and it will automatically change the ghost. The user will be able to cycle through all four ghost by clicking at least 4 times any each of the boxes. The eyes of the ghost in the canvas will be randomly pointing either left or right direction, after the user clicks the box. If the user moves the pointer around the respecting boxes, the ghost's eyes will follow it.

The message below the SVG and Canvas boxes will be displaying the current event that happened if the user tries to have an interaction with the boxes. (EXCLUDING FOLLOWING MOUSE POINTER)

Mouse click: Prints a message in message box.
