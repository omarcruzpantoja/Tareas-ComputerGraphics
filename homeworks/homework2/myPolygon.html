Functions of the demo
At loading a triangle will be drawn at the center of the canvas.

Mouse Click: If the user clicks inside the bounding box a new vertex will be created connected to the 2 closest points to the cursor pointer.
If the user clicks inside the circle of any vertex, it will be selected(highlitted with color yellow).
If the user clicks outside the bounding box, any selected vertex will be unselected.

Move mouse with click (Drag): If the user selects a vertex, the user might drag around the vertex(when the mouse is up, the vertex will stay in the last position).

Press Key arrows with vertex selected: If the user selects a vertex, he may press any key arrow to move the vertex + or - the pixels depending on the arrow pressed.

Press + with vertex selected: If a vertex is selected and + is pressed, a new vertex will be drawn in the mid point of the selected point and the next one.

Press - with vertex selected: The vertex will be deleted. A minimum of 3 vertices are required.
